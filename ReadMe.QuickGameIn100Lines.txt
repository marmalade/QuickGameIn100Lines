Using Template: Marmalade Quick Project

An example Marmalade Quick project using less than 100 lines of Lua code.
Touch anywhere to fire! Try to stop all the yellow/blue balls from reaching
the floor.

Works on any screen size or orientation.

Tested with Marmalade 7.5. Doesn't use any extensions.
Uses:
 - https://github.com/nickchops/MarmaladeQuickNodeUtility
 - https://github.com/nickchops/MarmaladeQuickVirtualResolution
